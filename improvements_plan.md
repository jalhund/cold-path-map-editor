# План улучшений

## 1. Предупреждение о слишком маленьких провинциях

Отображать предупреждения для провинций, которые слишком маленькие (малая площадь, ширина или высота), чтобы легче находить недочёты карты.

**Реализация:** На этапе экспорта/генерации, после подсчёта bounding box и площади каждой провинции, проверять минимальные пороги. Выводить предупреждения через msg.post + debug log:
```
WARNING: Province 15 is too small (area=42, width=3, height=14)
```

Нужно определить пороги: минимальная площадь, минимальная ширина, минимальная высота.

## 2. Замена всех пикселей определённого цвета на всей текстуре

Кнопка в редакторе провинций, которая заменяет ВСЕ пиксели одного цвета на другой по всей текстуре (не только связанную область как fill, а именно все пиксели этого цвета).

**Реализация:**
- Добавить функцию `replace_color(buffer_info, old_r, old_g, old_b, new_r, new_g, new_b)` в C++ (`drawpixels.cpp`) — проход по всем пикселям буфера, замена совпадающих по RGB.
- Добавить кнопку "Replace All" в UI редактора провинций.
- Порядок работы: пипеткой выбрать старый цвет -> выбрать/сгенерировать новый цвет -> нажать "Replace All".
